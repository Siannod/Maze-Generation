A python game I made using PyGame for my A Level final project.

This program utilises a variety of algorithms to create and solve random mazes.

MAZE CREATION
The mazes can be generated using either Prims or Depth First, these two demonstrate a large difference in the way the final maze appears. The player can then play the level while being timed to get on the highscore table.

MAZE SOLVING
The player can choose to have the computer solve the maze, choosing from Depth First Search, Breadth First Search and A*. Each step the solving algorithm takes will be displayed allowing for a clear demonstration of how each algorithm works.

UTILITY
The game allows the player to log in storing the log in details locally using an SQL database, the game also times the players solving attempts and stores the best times in additional SQL databases for the player to view. 

CONTROLS
Use the RMB to move through the menu and WASD to control the player in the maze
